## Yandex Practicum Projects

A collection of practical Data Science and Machine Learning tasks covering analytics, prediction models, time series, and numerical methods.

**Data Analysis (Online Store Strimchik).ipynb:** exploratory data analysis of an online store (user behavior, sales metrics, acquisition channels).  
**ML Predictions (HR Analytics).ipynb:** employee churn prediction using classification models.  
**Model Quality (One Click Store).ipynb:** supervised learning task comparing model performance and tuning hyperparameters.  
**Numerical Methods (Car Price).ipynb:** car price prediction using numerical methods and regression.  
**Time Series (Taxi).ipynb:** forecasting taxi demand using time series models.
